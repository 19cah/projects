How to install

It looks like you’re running macOS, Linux, or another Unix-like OS. To install a project, run the following in your terminal.

curl https://sh.abranhe.com/{project} -sSf | sh

To uninstall any project, run the following in your terminal 

curl https://sh.abranhe.com/{project}/uninstall -sSf | sh

It looks like you’re running Windows. At the moment we don't have any support for Windows, but you can still run it in a Docker container

Check in the project you'd like to intall for more information
